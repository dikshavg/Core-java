**Supercall state:
->Super call statement is used to call the Constructor of the parent class.
->Super call statement can only be used inside a constructor
->In a constructor if there is No this call statement we will always have super call statement

**we have 2 types of super call statement:
1)implicit super() statement
2)Explicit super() statement

1)implicit super() statement:
->whenever parent call is having a non-Argument Constructor then supercalll statement present in the sub class Constructor will be called implicitly
